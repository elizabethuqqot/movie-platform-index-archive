# movie-platform-index-archive
该项目围绕在线电影与电视剧平台的公开信息进行整理，对首页栏目分布、播放页标题形式及更新节奏进行归档说明。通过持续记录建立结构化资料库，便于对比不同影视网站的页面组织方式。仓库不包含视频文件，仅保存公开页面资料。
